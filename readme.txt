== Description ==

This plugin uses the envoimoinscher.com API to provide shipping costs and functionalities to WooCommerce:
-Select among 15 carriers which one you would like to show your clients
-Send using the EnvoiMoinsCher API and manage directly your shipments from the WooCommerce interface
-For the moment, only shipments to or from France are available

English and French translations are available.

== Installation ==

1. Download the plugin from http://ecommerce.envoimoinscher.com/module-plugin/woocommerce/
2. Ask for your API key if you don't have one
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Head to 'wp-admin/admin.php?page=envoimoinscher-settings' and configure the plugin

You can find help for how to setup the plugin on 'wp-admin/admin.php?page=envoimoinscher-settings&tab=help'
